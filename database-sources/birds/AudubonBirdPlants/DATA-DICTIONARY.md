# Data Dictionary: AudubonBirdPlants

**Source ID:** `BIRD-02` (deferred)
**Description:** National Audubon Society Native Plants Database. Deferred — requires JavaScript rendering and per-plant detail page scraping.
**Status:** Folder created, no data extracted yet.

## Notes

The Audubon database provides bird-friendly native plant recommendations by zip code. Each plant has:
- Scientific and common names
- Bird species it supports (food, shelter, nesting)
- Native range
- Light, soil, moisture requirements

To complete this dataset would require scraping ~443 plant detail pages per zip code using a headless browser (Selenium/Playwright).
