# Colorado State University Extension - Preventing Deer Damage

**Source:** Colorado State University Extension, Fact Sheet 6.520
**URL:** https://extension.colostate.edu/resource/preventing-deer-damage/
**Plants:** 55 species
**Region:** Colorado (mountain/foothill landscapes)

## About

This CSU Extension fact sheet provides deer browsing ratings for common Colorado landscape plants, organized by plant type (flowers, vines, trees and shrubs).

## Rating Scale (3 levels)

| Rating | Count | Description |
|--------|-------|-------------|
| Rarely browsed | 22 | Deer rarely eat this plant |
| Sometimes browsed | 16 | Deer occasionally browse |
| Often browsed | 17 | Deer frequently eat this plant |

## Data Fields

| Field | Description |
|-------|-------------|
| scientific_name | Binomial name |
| common_name | Common plant name |
| plant_type | Flowers, Vines, or Trees and Shrubs |
| deer_resistance | Often browsed, Sometimes browsed, or Rarely browsed |

## Files

- `plants.csv` - All 55 plants with ratings
- `plants.json` - JSON with metadata
- `plants.db` - SQLite database
- `scripts/build_data.py` - Data builder (embedded from web scrape)
