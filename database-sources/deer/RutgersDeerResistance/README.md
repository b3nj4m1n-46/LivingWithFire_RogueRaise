# Rutgers Cooperative Extension - Landscape Plants Rated by Deer Resistance

**Source:** Rutgers NJ Agricultural Experiment Station, Bulletin E271
**Authors:** Perdomo, P., Nitzsche, P., and Drake, D.
**URL:** https://extension.rutgers.edu/deer-resistant-plants
**Plants:** 326 unique species/cultivars
**Region:** Northern New Jersey

## About

This is one of the most widely referenced deer resistance plant lists in the US. The ratings were compiled with input from nursery and landscape professionals, Cooperative Extension personnel, and Master Gardeners in Northern New Jersey.

## Rating Scale

| Code | Rating | Description |
|------|--------|-------------|
| A | Rarely Damaged | Plants not preferred by deer |
| B | Seldom Severely Damaged | Minor damage occasionally |
| C | Occasionally Severely Damaged | Moderate damage in some seasons |
| D | Frequently Severely Damaged | Deer strongly prefer these plants |
