# Washington State University Extension - Deer Resistant Plants

**Source:** Washington State University Extension, Publication C063
**Adapted from:** Washington Department of Fish and Wildlife and Backyard Wildlife Sanctuary Program
**Plants:** 82 unique species
**Region:** Eastern Washington (mule deer and whitetail deer)
**Format Updated:** 2024

## About

This WSU Extension fact sheet provides a list of "best bets" for keeping deer disinterested in landscape plants. The list covers eastern Washington where mule deer and whitetail deer are browsers. The document notes that plant use by deer varies by population density, alternative food availability, winter weather, and regional taste preferences.

**Key caveat from source:** Trees must be protected when young. Fencing is still the only consistently effective tool for reducing deer damage.

## Rating Type

**Binary list** — plants listed are deer-resistant "best bets." No severity scale or rating levels.
