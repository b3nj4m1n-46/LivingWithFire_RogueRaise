# Federal Noxious Weed List

**Source:** USDA APHIS Plant Protection and Quarantine
**Effective:** December 10, 2010
**Plants:** 112 species (19 aquatic, 5 parasitic, 88 terrestrial)

## About

The Federal Noxious Weed List is maintained by USDA APHIS PPQ under authority of the Plant Protection Act. Plants on this list are prohibited from import or interstate movement without a permit. This is a key regulatory reference for identifying invasive plants.
