# Western Governors' Association - Top 50 Invasive Species in the West

**Source:** Western Governors' Association
**URL:** https://westgov.org/images/editor/WGA_Top_50_Invasive_Species_List_1.pdf
**Date:** December 2017
**Plants Extracted:** 26 (18 terrestrial, 8 aquatic) — plant species only from the Top 50

## About

WGA surveyed invasive species coordinators across western US states to create a ranked priority list. The original Top 50 includes animals, diseases, and algae — this extraction contains **plant species only**.
