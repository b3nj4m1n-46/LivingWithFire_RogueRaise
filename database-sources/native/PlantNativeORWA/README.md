# PlantNative.org - Western Oregon & Western Washington Native Plants

**Source:** PlantNative.org
**URL:** https://plantnative.org/rpl-orwa.htm
**Plants:** 59 native species
**Region:** Western Oregon and Western Washington

## About

PlantNative.org provides curated native plant lists by region for Pacific Northwest landscaping. This list covers western OR/WA natives with sun exposure, moisture needs, and mature height data.

## Data Fields

| Field | Description |
|-------|-------------|
| scientific_name | Binomial name |
| common_name | Common plant name |
| plant_type | Tree, Shrub, or Perennial |
| sun | F=Full, P=Part, S=Shade |
| moisture | W=Wet, A=Average, D=Dry |
| height | Mature height |
