# Pollinator Partnership - Pacific Lowland Guide

**Source:** Pollinator Partnership / NAPPC
**URL:** https://www.pollinator.org/guides
**Plants:** 28 pollinator-attracting species
**Region:** Pacific Lowland Mixed Forest Province (Oregon & Washington)

## About

Regional guide for selecting plants that attract pollinators. Includes bloom period, flower color, height, sun/soil requirements, pollinator types attracted, and host plant status for caterpillar/larval development.

## Data Fields

| Field | Description |
|-------|-------------|
| scientific_name | Binomial name |
| common_name | Common plant name |
| plant_type | Tree/Shrub, Perennial, or Vine |
| flower_color | Bloom color(s) |
| height_ft | Mature height in feet |
| bloom | Bloom season |
| sun | Light requirements |
| soil | Soil moisture/drainage |
| pollinators | Pollinator types attracted (bees, butterflies, hummingbirds, etc.) |
| host_plant | Whether plant serves as host for caterpillars/larvae |
