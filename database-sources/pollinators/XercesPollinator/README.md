# Xerces Society + Pollinator Partnership - Regional Pollinator Plant Lists

**Sources:** Xerces Society & Pollinator Partnership / NAPPC
**URL:** https://xerces.org/pollinator-conservation/pollinator-friendly-plant-lists
**Plants:** 428 pollinator-attracting species across 4 western regions
**Regions:** Maritime NW, CA Coastal Chaparral, CA Coastal Steppe/Redwood, Sierran Steppe

## About

Regional plant guides identifying native plants that attract pollinators (bees, butterflies, hummingbirds, beetles, flies, wasps). Includes bloom period, flower color, height, sun/soil requirements, and pollinator types attracted.

## Data Fields

| Field | Description |
|-------|-------------|
| scientific_name | Binomial name |
| common_name | Common plant name |
| region | Geographic region |
| source | Xerces Society or Pollinator Partnership |
| flower_color | Bloom color(s) |
| height | Mature height |
| bloom | Bloom season |
| sun | Light requirements |
| soil | Soil moisture/drainage |
| pollinators | Pollinator types attracted |
