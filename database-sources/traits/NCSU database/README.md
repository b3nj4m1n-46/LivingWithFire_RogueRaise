# NCSU Plant Toolbox Scraper

Scrapes the full plant database from the [NC State Extension Gardener Plant Toolbox](https://plants.ces.ncsu.edu/) — a collection of **4,680 plants** that grow in and around North Carolina, maintained by NC State University's Cooperative Extension Service.

## What's in the data

Each plant record includes up to **50+ fields** covering:

| Category | Fields |
|---|---|
| **Identity** | Scientific name, common names, synonyms, slug, phonetic spelling |
| **Taxonomy** | Genus, species, family |
| **Description** | Full text description |
| **Dimensions** | Height, width (as ranges, e.g. `5 ft. 0 in. - 8 ft. 0 in.`) |
| **Growth** | Plant type, habit/form, growth rate, maintenance, texture, life cycle |
| **Growing conditions** | Light, soil texture, soil pH, soil drainage, USDA hardiness zones, NC region, available space |
| **Flowers** | Color, bloom time, shape, petals, size, inflorescence, description |
| **Fruit** | Color, harvest time, type, length, width, description |
| **Leaves** | Color, fall color, type, arrangement, shape, margin, length, width, description |
| **Bark & Stems** | Color, surface, form, aromatic, description |
| **Landscape** | Location, theme, design feature |
| **Ecology** | Wildlife value, attracts (pollinators, birds, etc.), resistance |
| **Safety** | Poison severity, symptoms, toxic principle |
| **Other** | Origin, propagation strategy, play value, edibility, problems |
| **Cultivars** | Named cultivars/varieties with notes |
| **Tags** | Searchable keyword tags (1,100+ unique tags) |
| **Images** | Thumbnail image URLs |
| **Source URL** | Link back to the original plant page |

### Multi-value fields

Some fields can have multiple values per plant (e.g. a plant may have multiple flower colors or grow in multiple soil types). In the CSV, these are separated by ` | `. In JSON output they become arrays. In SQLite they are stored in a separate `plant_attributes` table.

## Output formats

All output files are saved to the `output/` directory.

### `plants.csv`

Flat spreadsheet format. Opens in Excel, Google Sheets, or any CSV reader. Multi-value fields are pipe-separated (` | `).

### `plants.json` (generated by converter)

Nested JSON array. Multi-value fields are proper arrays. Good for programmatic use.

### `plants.db` (generated by converter)

SQLite database with two tables:

- **`plants`** — One row per plant with all scalar fields as columns
- **`plant_attributes`** — Normalized table for multi-value fields (`plant_id`, `field`, `value`)

Indexed on `scientific_name`, `genus`, `family`, `plant_id`, and `field` for fast queries.

**Example queries:**

```sql
-- Find all plants in the Rosaceae family
SELECT scientific_name, usda_zones FROM plants WHERE family = 'Rosaceae';

-- Find all flower colors for a specific plant
SELECT value FROM plant_attributes
WHERE plant_id = 42 AND field = 'flower_color';

-- Count plants by family
SELECT family, COUNT(*) as n FROM plants GROUP BY family ORDER BY n DESC;
```

## How the scraper works

`ncsu_plant_scraper.py` does the following:

1. **Fetches the sitemap** at `https://plants.ces.ncsu.edu/sitemap.xml` to discover all plant page URLs
2. **Visits each plant page** one at a time with a 1.5-second delay between requests
3. **Parses the HTML** using BeautifulSoup, extracting data from `<dl>`/`<dt>`/`<dd>` definition list elements that the site uses consistently across all pages
4. **Saves progress** every 50 plants so work isn't lost if interrupted
5. **Supports resume** — if stopped (Ctrl+C), re-running picks up where it left off
6. **Retries failures** up to 3 times with exponential backoff

A full run takes approximately **2 hours** at 4,678 plants.

## How to use the converter

> **Note:** Update the `cd` path below to wherever you've placed this `PLANTSCRAPER` folder on your machine.

The CSV is the primary data source. To generate JSON and SQLite from it:

```bash
cd /path/to/PLANTSCRAPER
python3 convert_csv.py
```

This reads `output/plants.csv` and writes `output/plants.json` and `output/plants.db`. Takes a few seconds.

## How to re-run the scraper

If you need to re-scrape (e.g. the site has been updated with new plants):

> **Note:** Update the `cd` path below to wherever you've placed this `PLANTSCRAPER` folder on your machine.

```bash
cd /path/to/PLANTSCRAPER
pip install -r requirements.txt
python3 ncsu_plant_scraper.py
```

To start completely fresh, delete everything in `output/` first. Otherwise it will resume from where it left off.

## Dependencies

- Python 3.8+
- `requests`
- `beautifulsoup4`
- `lxml`

Install with:

```bash
pip install -r requirements.txt
```

The converter (`convert_csv.py`) uses only standard library modules and has no external dependencies.
